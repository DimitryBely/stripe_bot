Для оплаты отправьте данные карты через |:
Тип списания
Amount
Name
Card number
Month
Year
CVV
Email
Phone
Description
Statement description
Country
City
State
Address line 1
Address line 2
ZIP
Api-key

При отсутствии какого-либо параметра введите "-"
________________
Для смены прокси отправьте сообщение боту в следующем формате:

socks5|1.1.1.1:1111
________________
Для отключения прокси отправьте боту следующую команду:

"proxy stop", либо "proxy -s", либо "proxy off"
________________
Для смены стандартного API-ключа отправьте новый ключ, либо проведите транзакцию используя необходимый ключ
________________
Типы списания:
PreAuth - создаёт пользователя, прикрепляет к нему карту, затем проводит транзакцию

Charge - сразу проводит транзакцию. При использовании этой транзакции "email" и "phone number" не используются
